# Gusto Meets — Web Discovery Portal

Gusto Meets is an on-demand, peer-to-peer marketplace for renting residential 
terrace spaces by the hour for social activities (meetups, casual hangouts, etc.)

This repo contains the **Web Discovery Portal prototype** — a learning project 
to practice Git/GitHub collaboration before moving to the Flutter codebase.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)
